# Moteur de Simulation Financière et Analyse de Risque

Projet de programmation en langage C réalisé dans le cadre de la L1 MIASHS - Économie-Gestion à l'Université Paris Nanterre.

## Description
Ce programme en console utilise l'algorithme de Monte-Carlo pour évaluer les risques liés à la fluctuation des taux d'intérêt sur un portefeuille de prêts bancaires (gestion dynamique de la mémoire via des listes chaînées).

## Contenu du dépôt
* `main.c`, `finance.c`, `finance.h` : Code source du simulateur.
* `Rapport_Projet_Lucas_Skhiri.pdf` : Rapport détaillé présentant l'architecture, la répartition du travail assisté par IA et les résultats d'exécution.
* `Latex pour rapport de projet L1.zip` : Fichiers sources LaTeX du rapport.
